# Finance Sentinel 💰

An AI-powered personal finance management platform designed to help users track expenses, monitor financial activities, analyze spending patterns, and gain smart financial insights through an interactive and modern dashboard experience.

Built using React, TypeScript, Firebase, Firestore, Tailwind CSS, Vite, and AI-assisted development workflows.

## 📖 Project Story

Finance Sentinel started as an exploration into AI-assisted software development and modern financial technology solutions. The main goal of the project was to build a practical finance-focused platform capable of helping users manage expenses, monitor spending habits, and simplify financial tracking through a clean and interactive dashboard system.

The project was explored through AI-assisted workflows and later customized locally by integrating Firebase Authentication, Firestore Database, and improving the frontend architecture using React, TypeScript, Tailwind CSS, and Vite.

During development, I worked on Firebase configuration, environment variable setup, dependency management using pnpm, deployment workflows, frontend optimization, and debugging several real-world runtime/build issues. Multiple problems related to Firebase API configuration, Vite builds, package dependencies, deployment setup, and frontend rendering were analyzed and resolved throughout the development process.

AI-assisted coding and prompt engineering techniques were also used during development for debugging, troubleshooting, feature enhancement, UI improvements, and improving development efficiency.

More than just a finance tracker, Finance Sentinel became a hands-on learning experience in frontend engineering, Firebase integration, deployment workflows, debugging strategies, cloud technologies, and modern AI-assisted full-stack development practices.
